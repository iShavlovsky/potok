подробнее об индексах
POTOK
индекс Потока, рассчитывается помесячно по формуле: <Индекс предыдущего месяца> * (1 + <Доходность текущего месяца>). Доходность текущего месяца: (<Процентный доход за месяц> + <Возвраты из дефолтов> - <Дефолты>) / <Ср. портфель инвестиций>.
RUEYBCSTR
индекс облигаций повышенной доходности Московской биржи, включает в себя облигации из сектора повышенного инвестиционного риска.
MOEX
индекс Московской биржи, представляет собой ценовой, взвешенный по рыночной капитализации фондовый индекс, включающий 50 наиболее ликвидных акций крупнейших российских компаний.
RGBITR
индекс государственных облигаций Московской биржи, включает наиболее ликвидные облигации федерального займа.
JL
индекс доходности инвесторов ближайшего конкурента Потока, значения взяты с официального сайта.
